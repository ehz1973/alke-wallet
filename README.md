# ALKE-WALLET

## Breve descripción
------------------

ALKY Wallet es una aplicación web simple para gestionar operaciones básicas de una billetera digital. Permite a los usuarios iniciar sesión, consultar su historial de transacciones, depositar y enviar dinero de forma rápida y segura en una interfaz clara y responsiva.

## Tecnologías utilizadas
----------------------

- HTML5
- CSS3
- JavaScript (vanilla)

## Estructura principal del proyecto
-------------------------------

- `index.html` — Página principal / inicio.
- `login.html` — Formulario de inicio de sesión.
- `sendmoney.html` — Interfaz para enviar dinero.
- `deposit.html` — Interfaz para depósitos.
- `transactions.html` — Listado de transacciones.
- `assets/css/style.css` — Estilos del proyecto.
- `assets/js/` — Scripts del proyecto (`login.js`, `sendmoney.js`, `sendmoney2.js`).

## Principales funcionalidades
--------------------------

- Autenticación básica de usuario (página de login).
- Envío de dinero entre cuentas o destinatarios.
- Registro y visualización de transacciones.
- Depósitos simulados en la cuenta del usuario.
- Interfaz responsiva y validaciones básicas en formularios.
